Simulador de Fila de Impressão utilizando uma lista dinâmica em Java.

Compile os arquivos do projeto usando um ambiente de desenvolvimento Java ou o terminal. Execute a classe FilaDeImpressao.java para iniciar o programa. Siga as opções do menu para interagir com o Fila de Impressão.

Na opção 1 descrito Adicionar Trabalho de Impressão para adicionar o documento e a página, na opção 2 na descrição Remover Trabalho onde é removido o documento e depois mostra atualização da fila de impressão para ser imprimindo, na opção 3 com menu escrito Mostrar Fila de Impressão tamanho da fila para ser imprimindo no trabalho, na opção 4 é a saída do programação no terminal.
